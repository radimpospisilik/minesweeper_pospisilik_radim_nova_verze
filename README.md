# Minesweeper – Hledání min

## 📌 Popis projektu
Tato aplikace je jednoduchá desktopová hra „Hledání min“ vytvořená v jazyce C# pomocí Windows Forms.  
Cílem hry je odkrýt všechna bezpečná políčka bez kliknutí na minu.

## 🎮 Funkce aplikace
- Generování hrací plochy
- Náhodné rozmístění min
- Více obtížností (Základní, Jednoduchá, Těžká, Vlastní)
- Možnost nastavení vlastní velikosti pole (max. 5x5)
- Ukládání skóre do JSON souboru
- Zobrazení výsledků v přehledné tabulce (DataGridView)
- Časovač a ProgressBar
- Klávesové zkratky:
  - Ctrl + N → nová hra
  - Ctrl + S → uložení

## ⚙️ Požadavky na spuštění
- Operační systém Windows
- .NET Runtime (např. .NET 6 nebo novější)

## ▶️ Spuštění aplikace
1. Otevřete složku s aplikací
2. Spusťte soubor.
3. Pokud aplikace nejde spustit, nainstalujte .NET Runtime
4. - Soubor se nachází ve složce aplikace (bin/Debug)

## 🧠 Použité technologie
- C#
- Windows Forms
- JSON (System.Text.Json)
